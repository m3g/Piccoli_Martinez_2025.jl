Description

This repository contains supporting information and code for the paper "Role of Imidazolium Cation Hydrophobicity in Ubiquitin Solvation by Aqueous Ionic Liquids". The data includes simulations, plots, and tables used for comparison between ionic liquids containing [EMIM]+ and [BMIM]+ cations and their interactions with various anions.
